What is Blockchain?

A blockchain is a computer file or database(a collection of data stored on a computer system) that in this case open and distributed, specifically use to store data. A blockchain is a simple way of passing information from A to B in a secured and computerized manner.

Blockchain seems quite complicated from the technical side, but its core concept is really simple. In the blockchain concept, the data is distributed or shared across many systems, and the whole blockchain is decentralized. This decentralization concept of data is the heart of the blockchain, it means that no one has control over it.

Blockchain concept:

Blockchain is a Distributed Ledger Technology(DLT) that allows data to be stored globally on multiple servers. The blockchain stores the data and the information in the form of group also known as blocks or nodes. Each block is being connected with other blocks forming a chain termed as Blockchain, and hence each block stores a full record of data in it can be called a data-chain. A blockchain is secured and bound to each other using cryptographic principles.



Decentralization:

Decentralization is not an unusual concept. In building a network architecture for a technology-based solution three concepts or architecture are being considered: Distributed, Centralized, and Decentralized. As discussed above a blockchain technology is based on Decentralized network architecture.

A decentralization network allows the transfer of control and decision making from a centralized entity( organization, group, or single person) to a distributed network. This network develops a secure environment in which management and access to a data resource in an application or a system can be achieved greater and fairer.

Transparency:

As the nature of blockchain is decentralized all the transactions can be viewed transparently by either having a personal node or by using blockchain explorers that allow entities to see transactions occurring live. The nodes or blocks of blockchain have their individual copy of the chain that gets updated once fresh blocks are added to a chain. This feature of transparency allows us to track transactions whenever required.

Working of Blockchain:

To understand how blockchain technology works let's consider an example and suppose that there is person A and person B, A wants to send money to B using blockchain. The money is represented electronically as a ‘block’, this block is broadcasted and viewed by every third party involved in this network and once all these entities approve that transaction is valid this block of money then can be added to a chain that provides a lasting and transparent record of transactions and then finally the money transferred from A to B.
